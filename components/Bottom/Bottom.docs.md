# Bottom

사이드바 하단 영역 합성. Figma `Sidebar Items > Bottom` 컴포넌트.

`InfoCard(lg)` (선택) + `Additional(caption)` + `CellItem` 다수 (선택) + `User` (선택) 의 수직 스택.

## Token Usage
- `spacing.scale.16`
- `spacing.scale.24`
- `spacing.primitive.256`

## Behavior
- 각 섹션 boolean prop 으로 토글
- `card`, `cells`, `user` props 로 자식에 데이터 전달
